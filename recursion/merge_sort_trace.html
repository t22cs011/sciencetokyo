マージソートの再帰呼び出しトレース

`mergesort(a, 0, 4, w)` の呼び出しを「ボックス法」で可視化します。

mergesort(0, 4)

begin = 0, end = 4, mid = 2

1行目に出力: (0, 4)
左側を呼び出し中...
mergesort(0, 2)

begin = 0, end = 2, mid = 1

2行目に出力: (0, 2)
左側を呼び出し中...
mergesort(0, 1)

begin = 0, end = 1, mid = 0

3行目に出力: (0, 1)
左側を呼び出し中...
mergesort(0, 0)

begin = 0, end = 0

4行目に出力: (0, 0)
ベースケース (begin < end が偽)。終了し、呼び出し元に戻る。✔
左側終了。右側を呼び出し中...
mergesort(1, 1)

begin = 1, end = 1

5行目に出力: (1, 1)
ベースケース。終了し、呼び出し元に戻る。✔
両側終了。マージ処理後、呼び出し元に戻る。✔
左側終了。右側を呼び出し中...
mergesort(2, 2)

begin = 2, end = 2

6行目に出力: (2, 2)
ベースケース。終了し、呼び出し元に戻る。✔
両側終了。マージ処理後、呼び出し元に戻る。✔
左側終了。右側を呼び出し中...
mergesort(3, 4)

begin = 3, end = 4, mid = 3

7行目に出力: (3, 4)
...この後も同様に `m(3,3)` と `m(4,4)` が呼ばれる...
